# 郑航指南 (ZUA Guide)

> ✈️ 郑州航空工业管理学院龙子湖校区 · 准大一新生信息搜索平台

## 关于本站

这是一个为郑州航空工业管理学院（即将更名郑州航空航天大学）准大一新生打造的校园信息搜索网站。涵盖宿舍、食堂、图书馆、专业、分数线、交通、军训、社团、校园环境等全方位校园资讯。

**本站由郑航24级人工智能专业学长倾心整理。**

## 功能特色

- 🔍 **智能中文搜索** — 支持自然语言查询，如"宿舍有没有空调"、"食堂什么好吃"
- 📂 **分类浏览** — 按主题快速找到关心的内容
- 📱 **响应式设计** — 手机/平板/电脑都能流畅使用
- 🔗 **分享链接** — 搜索结果可通过URL直接分享（如 `?q=分数线`）
- 📝 **详情弹窗** — 点击结果卡片查看完整信息

## 技术栈

- 纯静态网站（HTML + CSS + JavaScript）
- 零后端依赖，数据存储为 JS 文件
- 部署于 GitHub Pages（免费）

## 本地运行

```bash
# 任意静态文件服务器即可
python -m http.server 8080
# 然后打开 http://localhost:8080
```

或直接双击 `index.html` 在浏览器中打开。

## 数据来源

信息搜集自：
- 郑州航空工业管理学院官方网站 (zua.edu.cn)
- 学校官方微信公众号
